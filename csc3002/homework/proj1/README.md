## 2. About MNIST Dataset and Conversion

This is the MNIST dataset of handwritten digits taken from [here](http://yann.lecun.com/exdb/mnist/).

These files are not in any standard image format, and I use a python script to convert it into `.png` format, which is saved in `../test_images` and `../train_images`. 

The python script is also saved as `../test_process` and `../train_process`.
